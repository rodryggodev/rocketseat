Document Object Model


Dom serve para manipular os elementos da tag html via id, class, e name. aqui vamos trabalhar direto com a dom.


let inputElement = document.getElementById('nome'); -> buscanto pelo id da tga, pois o id ẽ unico para casa id

let inputElement = document.getElementByTagName('input'); -> buscando pela diretamentepela tag

let inputElement = document.getElementByClassName('nome');-> buscando pela nome da class lembrando que ela retonar um array, pois pode ter mais de uma class com o mesmo nome no html.

let inputElement = document.querySelector('nome')-> o mais usado busca os elementos na dom,ele deixa a gente percorrer na dom ate achar o elemento. Podemos tamém pega os atributos do html let inputElement = document.getquerySelector(' input [nome-nome]'). podemos passar mais de um elemento usando o getquerySelectorAll();
